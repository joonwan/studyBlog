## 반복문

![[Pasted image 20231004134343.png]]

- 동일한 내용을 특정 횟수만큼 반복 처리할 때 사용
- 일정한 코드를 지정한 횟수만큼 자동으로 반복 실행 가능
- 코드 길이 단축
- 단순하고 명확한 코드
- 유지관리 용이

## 반복문 미사용

```C++
printf("1번\n");
printf("2번\n");
printf("3번\n");
printf("4번\n");
printf("5번\n");
```

## 반복문 사용

```C++

for(int i = 1; i<=5 ;i ++){
	printf("%d 번",i);
}
```

## 누적 합

```C++
int sum = 0;

for(int i=1; i<=5; i++){
	sum += i;
}
```
